# Güvenlik Politikası

Bir güvenlik açığı bulursanız token, kişisel veri veya çalışır exploit içeren ayrıntıları herkese açık
bir issue içinde paylaşmayın. Depoda GitHub Security Advisories etkinse özel raporlama özelliğini
kullanın; değilse önce bakım sorumlusundan güvenli bir iletişim kanalı isteyin.

Bir Discord bot tokeni yanlışlıkla paylaşıldıysa, dosyayı silmek tek başına yeterli değildir. Discord
Developer Portal üzerinden tokeni hemen yenileyin ve eski tokenin kullanıldığı ortamları kontrol edin.

Desteklenen çalışma zamanı `package.json` içindeki `engines.node` alanında belirtilir. Güvenlik
güncellemesi almayan Node.js sürümleri desteklenmez.
